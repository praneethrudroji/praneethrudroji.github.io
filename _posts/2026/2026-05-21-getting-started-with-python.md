## Introduction

Python is one of the most popular programming languages in the world. Known for its simplicity and versatility, Python is an excellent choice for beginners and experienced developers alike. This article will guide you through the basics of Python programming.

## Why Learn Python?

- **Ease of Use**: Python's syntax is simple and readable.
- **Versatility**: Used in web development, data science, AI, and more.
- **Community Support**: A large and active community provides extensive resources.
- **Career Opportunities**: High demand for Python developers in the job market.

## Setting Up Python

1. **Download and Install**: Visit [python.org](https://www.python.org) to download the latest version.
2. **Choose an IDE**: Popular options include PyCharm, VS Code, and Jupyter Notebook.
3. **Verify Installation**: Run `python --version` in your terminal to confirm.

## Writing Your First Python Program

Open your IDE or a text editor and write the following code:

```python
print("Hello, World!")
```

Save the file as `hello.py` and run it using the command:

```bash
python hello.py
```

## Python Basics

### Variables and Data Types

```python
name = "John"  # String
age = 25        # Integer
height = 5.9    # Float
is_student = True  # Boolean
```

### Control Structures

#### If-Else

```python
if age > 18:
    print("Adult")
else:
    print("Minor")
```

#### Loops

```python
for i in range(5):
    print(i)

while age > 0:
    age -= 1
    print(age)
```

### Functions

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))
```
